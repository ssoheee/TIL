# Git 사용법

Git 사용 및 명령어 정리



## I.  Git 명령어 순서

```
git init
git add 파일명
git commit -m "넣을 메세지"
git remote add origin 주소(https://github.com/ssoheee/TIL.git)
git push -u origin master
```

## II. 이미 만들어진 프로젝트에 추가하는 순서

git add [파일명]

git add . (안들어간 모든 파일들 다 들어감)

git commit -m "메세지"

git push -u origin master
